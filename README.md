🏨 Hotel Energy Optimizer

  Energy Efficiency Optimization using Hotel Booking Demand Prediction

A data science web application that predicts hotel booking cancellations 
using machine learning and calculates potential energy savings, cost 
reductions, and CO2 emission reductions.


 🌐 Live Application
Access the live app here:
[Hotel Energy Optimizer](https://hotel-energy-optimizer-325bec2zb6dhxc8zkppnyd.streamlit.app/)


 📋 Project Overview

Hotel Energy Optimizer is an AI-Based Energy Decision Support System 
that helps hotel managers reduce energy consumption by predicting 
which bookings will be cancelled and recommending smart energy 
saving actions.


🚀 Features

 📊 **Dashboard** — Interactive booking trends and cancellation analytics
 🤖 **Prediction** — ML-powered cancellation risk prediction
 ⚡ **Energy Calculator** — Calculate KWh, money and CO2 savings
 📈 **Model Evaluation** — Compare 3 ML models with confusion matrix
 🗺️ **Development Journey** — Project timeline from March to May 2026
 🔋 **Energy Action Plan** — AI energy recommendations for any occupancy
